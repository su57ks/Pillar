# Pillar
Это бесплатный мессенджер, с возможностью поднять свой сервер. По всем вопросам пишите в Telegram @su57ks

## Как использовать:
1) Установите Python, если еще нет
2) Скачайте из файлов при последнем релизе server.py и приложение для вашей ОС
3) Запустите server.py
4) Запустите приложение
5) Наслаждайтесь использованием!
