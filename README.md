# Pillar
Это бесплатный мессенджер, с возможностью поднять свой сервер. По всем вопросам пишите в Telegram [@su57ks](t.me/su57ks) или в [ишью репозитория](https://github.com/su57ks/Pillar/issues)

## Как использовать:
1) Установите Python, если еще нет
2) Скачайте из файлов при [последнем релизе](https://github.com/su57ks/Pillar/releases/latest) server.py и приложение для вашей ОС
3) Запустите server.py
4) Запустите приложение
5) Подключитесь к IP 127.0.0.1 и порту 8888
6) Наслаждайтесь использованием!
